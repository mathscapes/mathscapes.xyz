mxyz
----
Parent website of Mathscapes: https://mathscapes.xyz

Created with HTML and SCSS.

Fonts used:
- Montaga
- Chivo
- Inconsolata

Tilt effect created with Vanilla Tilt at https://micku7zu.github.io/vanilla-tilt.js/.

To compile SCSS
$ brew install scss
$ sass --watch assets/scss/style.scss assets/css/style.css
